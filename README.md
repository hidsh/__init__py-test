# __init__py-test

An Example of using packages and `__init__.py` in Python

# Folder tree

```
$ tree .
.
|-- main.py
|-- my_package
|   |-- __init__.py
|   `-- my_module.py
`-- tests.py
```


# Calling function in a package from main.py
```
$ python main.py
hoge!!!
```
